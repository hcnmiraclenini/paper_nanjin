# 固定配方预测导出与置信区间（2026-07-01）

本实验用于补强 Strict RAMR-VE 主结果的可追溯性和不确定性披露。实验不重新选择 checkpoint、集成权重或尺度参数，只读取已经由验证集确定的固定配方：

- 权重文件：`../results/strict_no_leakage_ensemble/all_balanced_final_once/selected_weights.csv`
- 汇总文件：`../results/strict_no_leakage_ensemble/all_balanced_final_once/ensemble_summary.json`
- 导出脚本：`src/export_fixed_ensemble_predictions.py`
- 仓库内证据摘要：`docs/experiments/artifacts/strict_ramr_ve_fixed_ensemble_summary_20260701.json`
- 服务器完整输出目录：`../results/strict_no_leakage_ensemble/fixed_recipe_predictions/`

## 1. 无泄露口径

该脚本仅执行固定配方评估：

1. 从 `selected_weights.csv` 读取验证集已确定的 checkpoint 权重。
2. 从 `ensemble_summary.json` 读取验证集已确定的全局尺度 `1.018`。
3. 重新构造训练/验证/测试的时间顺序数据划分，`RobustScaler` 仍仅在训练集拟合。
4. 对验证集和测试集分别导出最终预测，不做任何测试集选模、权重搜索或尺度搜索。

## 2. 导出文件

| 文件 | 说明 |
| --- | --- |
| `validation_predictions.csv` | 验证集逐日期、逐方向真实值/预测值/绝对误差 |
| `test_predictions.csv` | 测试集逐日期、逐方向真实值/预测值/绝对误差 |
| `validation_predictions.npz` | 验证集预测和真实值数组 |
| `test_predictions.npz` | 测试集预测和真实值数组 |
| `fixed_ensemble_prediction_summary.json` | 指标、bootstrap 置信区间和低于原结果阈值的比例 |

## 3. 点估计结果

| 划分 | MAPE/% | MSE | MAE | 日期行数 | 目标变量数 |
| --- | ---: | ---: | ---: | ---: | ---: |
| 验证集 | 31.2612 | 0.032375 | 0.129684 | 211 | 20 |
| 测试集 | **18.5463** | **0.035640** | **0.138028** | 212 | 20 |

测试集点估计与 `all_balanced_final_once/ensemble_summary.json` 完全一致，说明固定配方可重建最终结果。

## 4. Bootstrap 不确定性

Bootstrap 以“日期行”为抽样单位，每次抽样保留同一日期的 20 个方向变量，避免将同一天不同方向客流误当作完全独立样本。Bootstrap 次数为 2000，随机种子为 42。

| 指标 | 点估计 | Bootstrap均值 | 95% CI |
| --- | ---: | ---: | --- |
| MAPE/% | 18.5463 | 18.5303 | [16.8796, 20.3743] |
| MSE | 0.035640 | 0.035647 | [0.028133, 0.043991] |
| MAE | 0.138028 | 0.138055 | [0.125860, 0.151379] |

## 5. 与原 MoE-Rail 阈值的 bootstrap 比例

原 MoE-Rail 阈值为 MAPE=19.41%、MSE=0.037700、MAE=0.144000。在 2000 次日期级 bootstrap 中：

| 条件 | 比例 |
| --- | ---: |
| MAPE 低于原 MoE-Rail | 83.10% |
| MSE 低于原 MoE-Rail | 69.80% |
| MAE 低于原 MoE-Rail | 81.50% |
| 三项同时低于原 MoE-Rail | 63.15% |

因此，论文可如实写为：Strict RAMR-VE 的测试集点估计在三项指标上均优于原 MoE-Rail 与现代强基线；日期级 bootstrap 显示这种优势具有较高稳定性，但置信区间仍反映有限测试窗口下的不确定性。不能将该结果表述为任意重采样、任意线路或任意时间粒度下都显著占优。
