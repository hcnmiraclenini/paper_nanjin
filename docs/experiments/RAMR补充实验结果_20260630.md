# RAMR补充实验结果（2026-06-30）

数据：`/root/data3/huangchanni/data`，846个共同日期，20个客流变量。  
设置：lookback=6，horizon=1，hidden_dim=512，batch_size=64，训练上限300 epoch。  
说明：测试指标均使用验证集最优 checkpoint 重新评估，未使用 `best_model_latest.pth`。

| 实验设置 | 最佳验证MAPE | 测试MAPE | MSE | MAE |
| --- | ---: | ---: | ---: | ---: |
| RAMR完整模型（scene + regime + robust） | 32.73 | 20.88 | 0.043960 | 0.151176 |
| 无regime路由 | 32.47 | 21.37 | 0.041662 | 0.151356 |
| 无场景门控 | 32.75 | 22.44 | 0.048092 | 0.161865 |
| 统计专家basic（mean/std/max） | 33.24 | 19.74 | 0.040482 | 0.148277 |
| 统计专家quantile | 29.79 | 19.96 | 0.039693 | 0.148368 |
| 方差负载均衡 | 32.95 | 20.91 | 0.045297 | 0.154411 |

结论：

- scene gating 对测试集最明显：完整RAMR较无场景门控降低 1.56 个百分点。
- regime routing 有正贡献：完整RAMR较无regime路由降低 0.49 个百分点。
- 统计特征复杂度并非越高越好：basic 与 quantile 在本轮测试集上优于 robust。论文中应如实表述为“鲁棒分布偏移特征增强机制解释，但当前样本规模下未取得最低测试误差”。
- 门控解释图已生成到 `../results/paper_figures_ramr_full/`；北京南_from_nj 的波动率-短期权重 Pearson r=0.036，p=0.601，不支持显著线性相关结论。
