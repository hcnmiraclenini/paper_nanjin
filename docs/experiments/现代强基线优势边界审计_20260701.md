# 现代强基线优势边界审计（2026-07-01）

## 审计口径

- 本分析不训练模型、不选择 checkpoint、不调整集成权重。
- Strict RAMR-VE 使用已经固定的最终测试集预测文件；现代强基线使用已复核的测试集点估计。
- bootstrap 单位为测试集日期行，每次抽样保留该日期的 20 个目标变量，避免拆散同一日期的多变量结构。
- 由于现代强基线未保存逐样本预测，本分析是相对点估计的非配对优势边界审计，不写作配对显著性检验。

## Strict RAMR-VE 点估计与不确定性

- 点估计：MAPE=18.5463%，MSE=0.035640，MAE=0.138028。
- 2000 次日期级 bootstrap 95% CI：MAPE [16.8796, 20.3743]，MSE [0.028133, 0.043991]，MAE [0.125860, 0.151379]。

## 写作建议

- 与当前最强现代基线 FreEformer 相比，Strict RAMR-VE 点估计 MAPE/MSE/MAE 分别降低 7.98%、10.96%、6.83%。
- 以 FreEformer 点估计为阈值，日期级 bootstrap 中三项指标同时更优的比例为 82.90%；其中 MAPE、MSE、MAE 分项更优比例分别为 95.80%、85.20%、92.60%。
- 论文可写“点估计三指标均优于现代强基线，且相对最强基线的重采样优势概率较高”；不宜写作“已完成配对显著性检验”或“所有重采样均显著优于”。
