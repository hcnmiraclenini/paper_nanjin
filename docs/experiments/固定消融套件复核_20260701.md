# 固定消融套件复核（2026-07-01）

本实验用于补强审稿人提出的“日历特征输入、负载均衡约束、分布偏移特征工程与 regime 路由”消融证据。实验只评估已经由训练验证集保存的 `best_model_latest.pth`，不根据测试集重新选择 checkpoint、权重或尺度。

## 1. 实验口径

- 数据：`data/from_nj.csv` 与 `data/to_nj.csv`，内连接后 846 个共同日期、20 个双向客流变量。
- 划分：前 50% 训练、中间 25% 验证、后 25% 测试。
- 预处理：`RobustScaler` 仅在训练集拟合，再变换验证集和测试集。
- 脚本：`src/evaluate_fixed_ablation_suite.py`
- 服务器完整输出：`../results/fixed_ablation_suite_20260701/`
- 仓库内证据：
  - `docs/experiments/artifacts/fixed_ablation_metrics_20260701.csv`
  - `docs/experiments/artifacts/fixed_ablation_metrics_20260701.json`

## 3. 面向审稿意见的解释

1. 日历场景输入有效：移除 scene gating 后测试 MAPE 从 21.6706% 升至 23.3886%，MSE 与 MAE 也同步退化，说明可预知日历条件对路由有实质贡献。
2. Regime 路由有效：移除 regime routing 后测试 MAPE 升至 22.9143%，MAE 升至 0.160740；虽然 MSE 略低于完整单 checkpoint，但综合误差退化，支持窗口级非平稳状态编码的必要性。
3. 负载均衡约束形式影响较小：方差型负载均衡的 MAPE 接近完整模型，但 MAE 略差，说明主实验采用高熵负载均衡更利于稳定专家协同。
4. 分布偏移特征工程不是“越复杂越好”：quantile 变体在单 checkpoint 中表现最强，说明分位结构对南京南站数据最关键；robust 特征的价值主要体现在机制解释和与其他训练目标形成互补，最终 Strict RAMR-VE 的验证集集成也确实选入了 `stat_quantile`、`stat_basic` 与 `mae_robust` 系列 checkpoint。

## 4. 论文写法边界

论文中可以据此写：

- “日历场景输入和 regime routing 的移除均导致 MAPE/MAE 退化，支持场景条件化非平稳路由机制。”
- “分位数统计对当前数据集最有效，robust distribution shift expert 的定位是刻画窗口级分布偏移和提供可解释扰动代理，不等同于识别突发事件或极端天气成因。”
- “最终 Strict RAMR-VE 不是单 checkpoint 结果，而是验证集确定的固定集成；测试集三项指标为 MAPE=18.5463%、MSE=0.035640、MAE=0.138028，优于上述单 checkpoint 消融变体。”

不能写：

- “鲁棒统计特征必然优于 basic/quantile。”
- “窗口统计特征可以直接识别极端天气或突发事件类型。”
- “根据测试集重新选择了最优消融模型。”
