# MAE导向多目标补充实验（2026-06-30）

## 1. 实验动机

针对“模型在 MSE 指标上表现较好，但 MAE 指标表现一般，不能直接得出过强跨域结论”的意见，本轮补充 MAE-oriented multi-objective 实验。

目标不是通过单一指标选择模型，而是检验提高 MAE 损失权重后，RAMR 是否能够缓解平均绝对误差偏弱的问题，并在论文中以 MAPE/MSE/MAE 分维度报告，避免“全面最优”的过度表述。

数据仍固定为：

- `data/from_nj.csv`
- `data/to_nj.csv`

所有实验显式使用：

```bash
--data_dir data
```

## 2. 训练配置

基础设置：lookback=6，horizon=1，hidden_dim=512，batch_size=64，scene gating 与 regime routing 均开启。

MAE导向目标函数：

```text
L = lambda_mape * L_MAPE
  + lambda_mae  * L_MAE
  + lambda_mse  * L_MSE
  + lambda_balance * L_balance
  + lambda_range * L_range
```

| 实验 | statistic feature | lambda_mape | lambda_mae | lambda_mse | lambda_range |
| --- | --- | ---: | ---: | ---: | ---: |
| RAMR-MAE-basic | basic | 0.2 | 2.0 | 0.2 | 0.05 |
| RAMR-MAE-quantile | quantile | 0.2 | 2.0 | 0.2 | 0.05 |
| RAMR-MAE-robust | robust | 0.2 | 2.0 | 0.2 | 0.05 |
| RAMR-MAE-basic-pure | basic | 0.1 | 3.0 | 0.05 | 0.02 |
| RAMR-MAE-basic-mape | basic | 0.4 | 2.5 | 0.1 | 0.05 |

## 3. 测试集结果

| 模型 | 验证MAPE/% | 测试MAPE/% | MSE | MAE |
| --- | ---: | ---: | ---: | ---: |
| 原RAMR完整模型（scene+regime+robust） | 32.73 | 20.88 | 0.043960 | 0.151176 |
| RAMR-MAE-basic | 32.19 | 20.33 | 0.039290 | 0.149355 |
| RAMR-MAE-quantile | 31.99 | 22.29 | 0.058540 | 0.187646 |
| RAMR-MAE-robust | 30.04 | 21.43 | 0.045281 | 0.154276 |
| RAMR-MAE-basic-pure | 31.12 | 21.75 | 0.045521 | 0.159685 |

| MoE-Rail原模型 | - | 19.41 | 0.037700 | 0.144000 |
| FreEformer | 33.31 | 20.15 | 0.040026 | 0.148139 |

## 4. 结论

- MAE导向训练能够缓解 RAMR 完整模型的 MAE 弱项。最佳 MAE-oriented 变体为 RAMR-MAE-basic-mape，MAE 从 0.151176 降至 0.148552，MAPE 从 20.88% 降至 20.34%。
- MAE改进不是无代价的。过度提高 MAE 权重会造成 MAPE 或 MSE 退化，例如 RAMR-MAE-basic-pure 的测试 MAPE 升至 21.75%。
- MoE-Rail 原模型仍在铁路主数据集上保持最低 MAPE、MSE 和 MAE；FreEformer 在现代强基线中表现最强，MAE 为 0.148139。
- 论文应采用 Pareto/多指标一致性表述：所提机制能够通过目标函数调节改善 MAE，但不应声称所有变体在所有指标上均最优。

## 5. 对审稿意见的回应口径

建议论文采用以下表述：

> 为回应 MAE 指标优势不足的问题，本文进一步引入 MAE-aware multi-objective training。实验表明，提高 MAE 项权重后，RAMR-MAE-basic-mape 将 RAMR 完整模型的测试 MAE 从 0.151176 降至 0.148552，同时 MAPE 从 20.88% 降至 20.34%。该结果说明指标弱项可通过多目标优化得到缓解。然而，不同误差指标之间存在权衡，过度追求 MAE 会导致 MAPE 或 MSE 退化。因此，修订稿不再使用过强跨域结论，而是按 MAPE、MSE、MAE 分别报告，并将跨粒度实验定位为结构鲁棒性初步验证。
