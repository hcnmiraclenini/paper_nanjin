# 最终一致性核验（2026-07-01）

本文件用于将论文修订、实验结果、数据真实性和无泄露协议集中核对，避免不同报告、Markdown 稿和 Word 稿之间出现数值或表述不一致。

## 1. 数据真实性

主实验仅使用项目内真实南京南站双向日粒度客流数据：

| 文件 | 形状 | 缺失值 | 重复日期 | SHA256 |
| --- | ---: | ---: | ---: | --- |
| `data/from_nj.csv` | 854 x 11 | 0 | 0 | `c4f8919ce83c4c677d772036ed42fb5e76c99507f3368e2d32f081b556df4ae7` |
| `data/to_nj.csv` | 852 x 11 | 0 | 0 | `d16b3becfc12ed22a576741a505080e9dc279d4ab54d8c8ef597b729efb3c41e` |

两表按 `time` 字段内连接后得到 846 个共同日期和 20 个站点-方向客流变量。数据语义为“南京南站到其他站点”和“其他站点到南京南站”的日粒度客流量。

## 2. 无测试泄露协议

最终主结果采用 Strict RAMR-VE，证据文件为 `../results/strict_no_leakage_ensemble/all_balanced_final_once/ensemble_summary.json`。协议字段明确写明：

> Weights and scale are selected on validation set only; test set is evaluated once after selection.

核验要点：

- 时间顺序划分：训练集 423、验证集 211、测试集 212。
- `RobustScaler` 仅在训练集拟合。
- checkpoint、集成权重和尺度校准参数仅由验证集确定。
- 测试集只在最终方案固定后评估一次。
- `src/eval_all_checkpoints.py` 与 `src/pick_best_test_checkpoint.py` 已默认禁止测试集选模，legacy 测试集扫描必须显式加 `--allow_test_selection`。

## 3. 最终主结果

| 模型 | MAPE/% | MSE | MAE |
| --- | ---: | ---: | ---: |
| MoE-Rail 原论文结果 | 19.41 | 0.037700 | 0.144000 |
| Strict RAMR-VE | **18.5463** | **0.035640** | **0.138028** |

`beats_baseline` 中 MAPE、MSE、MAE 均为 `true`。因此在铁路主数据集上可以如实表述为三项指标均优于原 MoE-Rail。

固定配方预测导出摘要已保存为仓库内证据文件 `docs/experiments/artifacts/strict_ramr_ve_fixed_ensemble_summary_20260701.json`，完整逐样本预测位于服务器 `../results/strict_no_leakage_ensemble/fixed_recipe_predictions/`。该文件只读取验证集已确定的 checkpoint 权重和尺度，不重新搜索测试集。日期级 bootstrap（2000次，seed=42）给出的测试集 95% CI 为：MAPE [16.8796, 20.3743]、MSE [0.028133, 0.043991]、MAE [0.125860, 0.151379]。在 bootstrap 样本中，MAPE/MSE/MAE 低于原 MoE-Rail 阈值的比例分别为 83.10%、69.80%、81.50%，三项同时低于原阈值的比例为 63.15%。该结果用于不确定性披露，不能扩大为任意重采样均显著占优。

## 4. 现代强基线

现代强基线结果来自 `../results/modern_baselines_correct_data/modern_baselines_results.csv`：

| 模型 | 测试MAPE/% | MSE | MAE |
| --- | ---: | ---: | ---: |
| DLinear | 24.3430 | 0.054314 | 0.182450 |
| PatchTST | 24.3140 | 0.054858 | 0.170963 |
| iTransformer | 21.3711 | 0.043506 | 0.154333 |
| TimeMixer | 23.5480 | 0.042384 | 0.159879 |
| FreEformer | 20.1540 | 0.040026 | 0.148139 |
| Strict RAMR-VE | **18.5463** | **0.035640** | **0.138028** |

因此在当前南京南站主数据集和当前实验协议下，Strict RAMR-VE 的三项指标优于新增现代强基线。结论不得外推为所有线路、所有时间粒度或所有公开数据集上必然全面最优。

进一步的现代强基线优势边界审计保存在 `docs/experiments/现代强基线优势边界审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/modern_baseline_margin_audit_20260701.csv` 和 `docs/experiments/artifacts/modern_baseline_margin_audit_20260701.json`。该审计不重新训练、不重新选模、不调整集成权重，仅将固定 Strict RAMR-VE 测试预测的日期级 bootstrap 分布与现代强基线点估计比较。相对当前最强现代基线 FreEformer，Strict RAMR-VE 点估计 MAPE/MSE/MAE 分别降低 7.98%、10.96%、6.83%；以 FreEformer 点估计为阈值，2000 次日期级 bootstrap 中三项同时更优比例为 82.90%，分项 MAPE/MSE/MAE 更优比例分别为 95.80%、85.20%、92.60%。由于现代强基线未保存逐样本预测，该分析只能写作非配对优势边界审计，不能写作配对显著性检验。

站点方向误差剖面审计保存在 `docs/experiments/站点方向误差剖面审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/target_error_profile_20260701.csv` 和 `docs/experiments/artifacts/target_error_profile_20260701.json`。该审计只读取固定 Strict RAMR-VE 测试预测，不重新训练、不重新选模。结果显示，“南京南→方向站”和“方向站→南京南”两个方向的聚合 MAPE 分别为 17.8501% 和 19.2425%，MSE 分别为 0.032817 和 0.038224，MAE 分别为 0.136398 和 0.139621；20 个目标变量 MAPE 中位数为 16.4564%，16/20 个目标变量 MAPE 低于 20%，17/20 个目标变量低于 30%。成都东、广州南等低平均客流方向 MAPE 较高，论文需将其作为低流量方向相对误差局限披露。

流量分层误差审计保存在 `docs/experiments/流量分层误差审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/flow_stratified_error_audit_20260701.csv` 和 `docs/experiments/artifacts/flow_stratified_error_audit_20260701.json`。该审计只读取固定 Strict RAMR-VE 测试预测，不训练、不调参、不重新选模。结果显示，单点客流不低于 3000 人次样本的 MAPE/MSE/MAE 为 13.0327%/0.034273/0.129906，低流量样本（小于 500 人次）为 34.5288%/0.245261/0.334380；高需求日为 23.9754%/0.054716/0.187630，高于常规需求日的 16.1755%/0.021710/0.114732。该结果应写作误差分层与局限分析，不得作为新的选模依据。

## 4.1 固定消融套件复核

固定消融复核来自 `../results/fixed_ablation_suite_20260701/fixed_ablation_metrics.csv`，仓库内证据为 `docs/experiments/artifacts/fixed_ablation_metrics_20260701.csv` 和 `docs/experiments/artifacts/fixed_ablation_metrics_20260701.json`。该复核只评估验证集保存的 `best_model_latest.pth`，不按测试集重新选择 checkpoint。

| 变体 | 测试MAPE/% | MSE | MAE |
| --- | ---: | ---: | ---: |
| RAMR-full(scene+regime+robust+entropy) | 21.6706 | 0.046175 | 0.156578 |
| w/o scene gating | 23.3886 | 0.052137 | 0.171826 |
| w/o regime routing | 22.9143 | 0.044304 | 0.160740 |
| variance load balance | 21.7700 | 0.046378 | 0.158459 |
| distribution basic(mean/std/max) | 21.0201 | 0.042787 | 0.157434 |
| distribution quantile | 19.0447 | 0.035678 | 0.136987 |
| MAE-aware robust | 21.2186 | 0.049823 | 0.164273 |

该表支持三点写法：第一，移除日历场景门控和移除 regime routing 均导致 MAPE/MAE 退化；第二，quantile 分布特征是当前单 checkpoint 中最强的统计特征集合；第三，最终 Strict RAMR-VE 的 18.5463%/0.035640/0.138028 是验证集固定集成结果，优于上述单 checkpoint 消融变体。

## 7. 文本边界

论文与逐条说明应保持以下边界：

- 分布偏移专家只能刻画客流窗口 distribution shift，不能识别极端天气、突发事件或运营管制的具体因果来源。
- Traffic 小时级实验只能说明重构窗口、周期特征和扰动代理后的结构鲁棒性初步验证，不能证明日粒度铁路模型可直接跨分辨率迁移。
- Traffic 实验中 MoE-Rail 的 MSE 最优但 MAE 不是最优，因此不得表述为跨域泛化能力强或所有指标全面占优。
- 当前工作区没有可直接复现实验的 Traffic 原始数据文件；Traffic 数值不得表述为当前仓库已从原始数据重新训练复现，具体证据边界见 `docs/experiments/Traffic跨粒度证据边界_20260701.md`。
- Strict RAMR-VE 的优势限定在当前南京南站双向日粒度客流、当前时间顺序划分和当前对比模型集合下。
- MAE-aware 单模型训练能缓解 MAE 弱项，但最终三指标同步改善来自验证集选择的互补 checkpoint 集成与尺度校准，不是测试集调参。
