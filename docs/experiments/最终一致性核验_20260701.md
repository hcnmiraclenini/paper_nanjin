# 最终一致性核验（2026-07-01）

本文件用于将论文修订、实验结果、数据真实性和无泄露协议集中核对，避免不同报告、Markdown 稿和 Word 稿之间出现数值或表述不一致。

## 1. 数据真实性

主实验仅使用项目内真实南京南站双向日粒度客流数据：

## 2. 无测试泄露协议

最终主结果采用 Strict RAMR-VE，证据文件为 `../results/strict_no_leakage_ensemble/all_balanced_final_once/ensemble_summary.json`。协议字段明确写明：

> Weights and scale are selected on validation set only; test set is evaluated once after selection.

核验要点：

- 时间顺序划分：训练集 423、验证集 211、测试集 212。
- `RobustScaler` 仅在训练集拟合。
- checkpoint、集成权重和尺度校准参数仅由验证集确定。
- 测试集只在最终方案固定后评估一次。
- `src/eval_all_checkpoints.py` 与 `src/pick_best_test_checkpoint.py` 已默认禁止测试集选模，legacy 测试集扫描必须显式加 `--allow_test_selection`。

可执行协议审计保存在 `docs/experiments/无泄露实验协议审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/no_leakage_protocol_audit_20260701.json`。该审计重建真实 CSV 的 846 天共同序列和 20 个目标变量，验证训练/验证/测试日期区间互不重叠，确认当前 `RobustScaler` 参数与“仅训练集拟合”一致且不等于全量拟合，并逐样本检查输入窗口日期均早于预测日期；固定测试预测文件行数为 212，与测试集样本数一致。

## 3. 最终主结果

| 模型 | MAPE/% | MSE | MAE |
| --- | ---: | ---: | ---: |
| MoE-Rail 原论文结果 | 19.41 | 0.037700 | 0.144000 |

`beats_baseline` 中 MAPE、MSE、MAE 均为 `true`。因此在铁路主数据集上可以如实表述为三项指标均优于原 MoE-Rail。

固定配方预测导出摘要已保存为仓库内证据文件 `docs/experiments/artifacts/strict_ramr_ve_fixed_ensemble_summary_20260701.json`，完整逐样本预测位于服务器 `../results/strict_no_leakage_ensemble/fixed_recipe_predictions/`。该文件只读取验证集已确定的 checkpoint 权重和尺度，不重新搜索测试集。日期级 bootstrap（2000次，seed=42）给出的测试集 95% CI 为：MAPE [16.8796, 20.3743]、MSE [0.028133, 0.043991]、MAE [0.125860, 0.151379]。在 bootstrap 样本中，MAPE/MSE/MAE 低于原 MoE-Rail 阈值的比例分别为 83.10%、69.80%、81.50%，三项同时低于原阈值的比例为 63.15%。该结果用于不确定性披露，不能扩大为任意重采样均显著占优。

进一步的现代强基线优势边界审计保存在 `docs/experiments/现代强基线优势边界审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/modern_baseline_margin_audit_20260701.csv` 和 `docs/experiments/artifacts/modern_baseline_margin_audit_20260701.json`。该审计不重新训练、不重新选模、不调整集成权重，仅将固定 Strict RAMR-VE 测试预测的日期级 bootstrap 分布与现代强基线点估计比较。相对当前最强现代基线 FreEformer，Strict RAMR-VE 点估计 MAPE/MSE/MAE 分别降低 7.98%、10.96%、6.83%；以 FreEformer 点估计为阈值，2000 次日期级 bootstrap 中三项同时更优比例为 82.90%，分项 MAPE/MSE/MAE 更优比例分别为 95.80%、85.20%、92.60%。由于现代强基线未保存逐样本预测，该分析只能写作非配对优势边界审计，不能写作配对显著性检验。

现代强基线全指标支配审计保存在 `docs/experiments/现代强基线全指标支配审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/modern_baseline_dominance_audit_20260701.csv`、`docs/experiments/artifacts/modern_baseline_dominance_audit_20260701.json` 和 `docs/experiments/figures/modern_baseline_all_metrics_20260701.png`。该审计同样不训练、不调参、不重新选模。结果显示，Strict RAMR-VE 在 5 个现代强基线 x 3 个指标的 15 个点估计比较单元中全部更低；其中最接近边界仍是 FreEformer，MAPE/MSE/MAE 相对降幅为 7.98%/10.96%/6.83%。以 Strict RAMR-VE 日期级 95% CI 上界与各基线点估计比较，严格低于基线的单元为 10/15；因此论文可写“点估计全指标占优”，但不写作逐样本配对显著性检验。

现代强基线配对预测审计保存在 `docs/experiments/现代强基线配对预测审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/modern_baseline_paired_audit_20260701.csv`、`docs/experiments/artifacts/modern_baseline_paired_audit_20260701.json` 和 `docs/experiments/artifacts/modern_baseline_paired_predictions_20260701/`。该审计来自一次额外的 prediction-export rerun：现代强基线仍按验证集选择最佳 epoch，但额外保存测试集逐样本预测；审计脚本逐文件校验测试目标与 Strict RAMR-VE 固定预测完全一致。结果显示，在同一测试日期和20个目标变量的配对口径下，Strict RAMR-VE 对 DLinear、PatchTST、iTransformer、TimeMixer、FreEformer 的 MAPE/MSE/MAE 共 15/15 个点估计比较单元均更低；日期级 bootstrap 中三指标同时更优的概率范围为 89.40% 至 100.00%，最接近边界为 TimeMixer。该结果可作为逐样本可比性补充证据，但因其来自额外重跑，不替换主文原现代强基线表。

现代强基线配对统计审计保存在 `docs/experiments/现代强基线配对显著性审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/paired_significance_audit_20260701.csv` 和 `docs/experiments/artifacts/paired_significance_audit_20260701.json`。该审计以测试日期为配对单位，保留每个日期 20 个目标变量整体结构，不训练、不调参、不重新选模。结果显示，15/15 个点估计差值为正，15/15 个 Wilcoxon-Holm 单侧检验在 α=0.01 下显著；bootstrap 95% CI 下界为正和符号置换-Holm 显著的比较单元均为 14/15。唯一边界为 TimeMixer 的 MSE（bootstrap CI [-0.001809, 0.008071]，符号置换-Holm p=0.1057），因此论文可写“点估计和日期级配对秩检验均支持优势”，但仍需披露 TimeMixer-MSE 边界。

场景分层现代强基线审计保存在 `docs/experiments/场景分层现代强基线审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/scene_baseline_stratification_audit_20260701.csv` 和 `docs/experiments/artifacts/scene_baseline_stratification_audit_20260701.json`。该审计从真实 CSV 重建测试日期并校验目标数组与固定预测 artifact 完全一致，只用于工作日/周末场景误差刻画，不训练、不调参、不重新选模。结果显示，152 个工作日样本上 Strict RAMR-VE 的 MAPE/MSE/MAE 为 18.1153%/0.038406/0.135967，60 个周末样本上为 19.6382%/0.029723/0.142826；两类场景三项指标均优于 DLinear、PatchTST、iTransformer、TimeMixer 和 FreEformer。论文可据此写“日历场景特征和场景条件化门控在工作日/周末两类可检验场景下均有误差收益”，但不得外推节假日场景，因为当前测试窗口节假日样本为 0。

站点方向误差剖面审计保存在 `docs/experiments/站点方向误差剖面审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/target_error_profile_20260701.csv` 和 `docs/experiments/artifacts/target_error_profile_20260701.json`。该审计只读取固定 Strict RAMR-VE 测试预测，不重新训练、不重新选模。结果显示，“南京南→方向站”和“方向站→南京南”两个方向的聚合 MAPE 分别为 17.8501% 和 19.2425%，MSE 分别为 0.032817 和 0.038224，MAE 分别为 0.136398 和 0.139621；20 个目标变量 MAPE 中位数为 16.4564%，16/20 个目标变量 MAPE 低于 20%，17/20 个目标变量低于 30%。成都东、广州南等低平均客流方向 MAPE 较高，论文需将其作为低流量方向相对误差局限披露。

流量分层误差审计保存在 `docs/experiments/流量分层误差审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/flow_stratified_error_audit_20260701.csv` 和 `docs/experiments/artifacts/flow_stratified_error_audit_20260701.json`。该审计只读取固定 Strict RAMR-VE 测试预测，不训练、不调参、不重新选模。结果显示，单点客流不低于 3000 人次样本的 MAPE/MSE/MAE 为 13.0327%/0.034273/0.129906，低流量样本（小于 500 人次）为 34.5288%/0.245261/0.334380；高需求日为 23.9754%/0.054716/0.187630，高于常规需求日的 16.1755%/0.021710/0.114732。该结果应写作误差分层与局限分析，不得作为新的选模依据。

高需求高波动压力场景审计保存在 `docs/experiments/高需求高波动压力场景审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/event_stress_baseline_audit_20260701.csv` 和 `docs/experiments/artifacts/event_stress_baseline_audit_20260701.json`。该审计从真实 CSV 重建测试日期并校验目标数组与固定预测 artifact 完全一致，只用于压力场景误差刻画。结果显示，日总客流最高 25% 的高需求日上 Strict RAMR-VE 的 MAPE/MSE/MAE 为 23.9754%/0.054716/0.187630，三项均优于五个现代强基线；日际客流绝对变化最高 25% 的高波动日上 Strict RAMR-VE 的 MAPE 为 24.8108%，排名第一，但 MSE/MAE 最优模型为 TimeMixer。论文可据此写“高需求压力场景优势成立”，但不得写作统计特征已经识别或解决所有突发事件、极端天气、运营管制等外生扰动的因果来源。

Regime 门控对齐审计保存在 `docs/experiments/Regime门控对齐审计_20260701.md`，仓库内证据为 `docs/experiments/artifacts/regime_gate_alignment_audit_20260701.csv` 和 `docs/experiments/artifacts/regime_gate_alignment_audit_20260701.json`。该审计只加载论文机制模型 checkpoint，不训练、不调参、不重新选模；volatility、peak-to-average、trend strength 和 robust shift z 均由预测时可见的历史回看窗口计算。结果显示，日总波动率最高 25% 样本的短期专家权重为 0.6714，高于其余样本的 0.4256，p=2.58e-06；峰均比最高 25% 样本的分布偏移专家权重为 0.5909，高于其余样本的 0.3303，p=1.83e-05。该结果用于支持“不同非平稳 regime 对应不同专家”的机制解释，不得写成简单线性或因果规律。

## 7. 文本边界

论文与逐条说明应保持以下边界：

- 分布偏移专家只能刻画客流窗口 distribution shift，不能识别极端天气、突发事件或运营管制的具体因果来源。
- Traffic 小时级实验只能说明重构窗口、周期特征和扰动代理后的结构鲁棒性初步验证，不能证明日粒度铁路模型可直接跨分辨率迁移。
- Traffic 实验中 MoE-Rail 的 MSE 最优但 MAE 不是最优，因此不得表述为跨域泛化能力强或所有指标全面占优。
- 当前工作区没有可直接复现实验的 Traffic 原始数据文件；Traffic 数值不得表述为当前仓库已从原始数据重新训练复现，具体证据边界见 `docs/experiments/Traffic跨粒度证据边界_20260701.md`。
- Strict RAMR-VE 的优势限定在当前南京南站双向日粒度客流、当前时间顺序划分和当前对比模型集合下。
- MAE-aware 单模型训练能缓解 MAE 弱项，但最终三指标同步改善来自验证集选择的互补 checkpoint 集成与尺度校准，不是测试集调参。
