补充实验结果如实写入论文：在固定 checkpoint 复核中，basic（mean/std/max）测试 MAPE 为 21.0201%，quantile 为 19.0447%，robust/RAMR 单 checkpoint 为 21.6706%。因此修订稿不声称鲁棒统计必然带来最低单模型误差，而是强调 quantile 结构对当前数据最有效，robust distribution shift 特征用于提升分布扰动代理的机制解释，并在最终 Strict RAMR-VE 验证集集成中与 `stat_quantile`、`stat_basic` 系列形成互补。

### 4. 短期专家权重偏低如何解释

已重新核验 `2.7` 门控权重。重算发现旧版表格存在专家列顺序解释问题：当前模型输出专家顺序为 `ShortTerm / LongTerm / DistributionShift`，因此不能继续按“长期、短期、分布偏移”的旧列顺序解释。修订稿已据此替换表4、表5和对应文字。

严格测试集门控统计为：

- 工作日样本 152 个：短期专家 0.5927，长期专家 0.0845，分布偏移专家 0.3228。
- 周末样本 60 个：短期专家 0.2196，长期专家 0.2011，分布偏移专家 0.5793。
- 当前严格时间顺序测试窗口节假日样本数为 0，因此不再报告节假日 vs 工作日显著性检验，也不声称当前测试集上节假日权重抬升显著。

修订后的解释为：短期专家并非在所有场景中权重最低。工作日中短期专家权重最高，说明其承担相邻日期连续性和短程校正；周末中分布偏移专家权重显著升高，说明非工作日需求结构变化触发分布偏移建模。Welch t 检验显示，周末分布偏移专家权重显著高于工作日（0.5793 vs 0.3228，t=4.9757，p=2.42e-06）。该结论比旧版“短期专家整体低权重”的解释更符合当前模型真实路由行为。

同时保留波动率-短期权重图作为补充可视化。该图在 `北京南_from_nj` 上 Pearson r=0.0467、p=0.4990，未形成显著线性相关。修订稿据此避免过度声称“波动越大短期权重必然越高”，机制证据以场景门控均值、周末/工作日 Welch 检验和专家消融为主。

本轮进一步新增 Regime 门控对齐审计，避免门控解释只依赖场景均值或单站点散点图。该审计只加载论文机制模型 checkpoint，不训练、不调参、不重新选模；volatility、peak-to-average、trend strength、robust shift z 等指标均由预测时可见的历史回看窗口计算。结果显示：日总波动率最高 25% 的样本中短期专家权重均值为 0.6714，高于其余样本的 0.4256，p=2.58e-06；峰均比最高 25% 的样本中分布偏移专家权重均值为 0.5909，高于其余样本的 0.3303，p=1.83e-05。因此修订稿将机制解释改为“高频相邻日变化更多触发短期专家，峰均比突发强度更多触发分布偏移专家”，不再写成“所有波动都单调触发同一个专家”。

### 3. MSE 好但 MAE 一般，跨域泛化结论不充分

已重新实验并修改。除原有 RAMR 消融和现代强基线外，新增了 MAE-aware multi-objective 训练，并进一步补充严格无泄露验证集集成（Strict RAMR-VE）。本轮先修正数据预处理：`RobustScaler` 默认仅使用训练集拟合，再变换验证集和测试集，避免验证/测试分布泄露到训练预处理。随后在验证集上确定 checkpoint 集合、集成权重和尺度校准参数；候选表只导出验证集指标，不导出候选测试指标；测试集只在最终方案固定后评估一次。最终结果文件为 `../results/strict_no_leakage_ensemble/all_balanced_final_once/ensemble_summary.json`。

| 模型 | 测试MAPE/% | MSE | MAE |
| --- | ---: | ---: | ---: |
| 早期RAMR完整模型（MAE实验参照） | 20.88 | 0.043960 | 0.151176 |
| RAMR-MAE-basic | 20.33 | 0.039290 | 0.149355 |
| RAMR-MAE-basic-mape | 20.34 | 0.040598 | 0.148552 |
| MoE-Rail原模型 | 19.41 | 0.037700 | 0.144000 |
| FreEformer | 20.15 | 0.040026 | 0.148139 |
| Strict RAMR-VE | **18.55** | **0.035640** | **0.138028** |
| Strict RAMR-VE（验证集选权重+尺度） | **18.55** | **0.035640** | **0.138028** |

实验结论更新为：单一 MAE 权重调整可以缓解 RAMR 的 MAE 弱项，但仍存在 MAPE/MSE/MAE 权衡；进一步引入验证集集成后，Strict RAMR-VE 在严格无测试泄露口径下取得 MAPE=18.55%、MSE=0.035640、MAE=0.138028，相比原 MoE-Rail 的 19.41%、0.037700、0.144000 三项指标均改善。该改进不是测试集调参，而是由验证集 balanced objective 选出，测试集只用于最终一次评估。

为增强可追溯性，本轮又用固定配方导出验证集和测试集逐样本预测，并做 2000 次日期级 bootstrap。测试集点估计保持 MAPE=18.5463%、MSE=0.035640、MAE=0.138028；bootstrap 95% CI 分别为 MAPE [16.8796, 20.3743]、MSE [0.028133, 0.043991]、MAE [0.125860, 0.151379]。在 bootstrap 样本中，MAPE/MSE/MAE 低于原 MoE-Rail 阈值的比例分别为 83.10%、69.80%、81.50%，三项同时低于原阈值的比例为 63.15%。因此论文表述为“点估计三指标均优且重采样稳定性较高”，不写成任意重采样都显著占优。

为进一步回应“MAE 指标表现一般、是否仅 MSE 占优”的质疑，本轮新增现代强基线优势边界审计。该审计不重新训练、不重新选模、不调整集成权重，只使用已经固定的 Strict RAMR-VE 测试集预测，并与现代强基线点估计比较。相对当前最强现代基线 FreEformer，Strict RAMR-VE 点估计 MAPE/MSE/MAE 分别降低 7.98%、10.96%、6.83%；以 FreEformer 点估计为阈值，2000 次日期级 bootstrap 中三项同时更优比例为 82.90%，分项 MAPE/MSE/MAE 更优比例分别为 95.80%、85.20%、92.60%。由于现代强基线没有保存逐样本预测，该结果写作“非配对优势边界审计”，不写作配对显著性检验。

本轮进一步新增现代强基线全指标支配审计和三指标并列图。该审计仍只读取固定测试预测和现代强基线点估计，不训练、不调参、不重新选模。结果显示，Strict RAMR-VE 在 5 个现代强基线 x 3 个指标的 15 个点估计比较单元中全部更低，支配率为 15/15；最接近的 FreEformer 边界上，MAE 仍下降 6.83%。以 Strict RAMR-VE 日期级 95% CI 上界与各基线点估计比较，严格低于基线的单元为 10/15。论文据此写作“当前协议下点估计全指标占优”，同时保留“非配对、非全数据集泛化证明”的边界。

为避免聚合指标掩盖局部目标变量问题，本轮又新增站点方向误差剖面审计。Strict RAMR-VE 在“南京南→方向站”和“方向站→南京南”两个方向上的聚合 MAPE 分别为 17.8501% 和 19.2425%，MSE 分别为 0.032817 和 0.038224，MAE 分别为 0.136398 和 0.139621；20 个目标变量的 MAPE 中位数为 16.4564%，其中 16/20 个目标变量 MAPE 低于 20%，17/20 个目标变量低于 30%。同时如实披露成都东、广州南等低平均客流方向 MAPE 较高，原因是低客流小分母会放大百分比误差，后续可用分层损失和外生扰动变量继续改进。

为进一步解释 MAE/MAPE 的误差来源，本轮新增流量分层误差审计。该审计只读取固定 Strict RAMR-VE 测试预测，不训练、不调参、不重新选模。单点客流不低于 3000 人次的样本 MAPE/MSE/MAE 为 13.0327%/0.034273/0.129906，低流量样本（小于 500 人次）为 34.5288%/0.245261/0.334380，说明主业务高流量样本相对误差更稳定，而低流量小分母会显著放大百分比误差。按测试日期总需求分层时，高需求日 MAPE/MSE/MAE 为 23.9754%/0.054716/0.187630，高于常规需求日的 16.1755%/0.021710/0.114732，说明集中需求或峰值日期仍是后续优化重点。论文将该结果写作误差分层与局限分析，不作为新的选模依据。

论文表述已同步修改为：

- Traffic 结果用于说明结构鲁棒性初步成立，不证明日粒度铁路模型可直接跨分辨率迁移。
- 铁路主数据集最终主结果采用 Strict RAMR-VE，可写成 MAPE/MSE/MAE 三项指标均优于旧 MoE-Rail 与现代强基线。
- 结论中增加限制：稳定泛化仍需更多铁路线路、更多公开数据集和更多外生扰动变量验证。
